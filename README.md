Esto es un programa crud de un gestor de donaciones con las tablas, donante(Donor),
 beneficiarios(Beneficiaries) y donaciones(donation). en cada una de las tablas tenemos la
 opcion de crear(Add), editar(update), enlistar(Getall) y eliminar(Delete) esto automatizando
 la base de datos y facilitando la vida a los usuarios, la principal ventaja de este proyecto es
 que podras digitalizar la informacion de tu negocio o empresa sin los complejos codigos
 que se usan en las bases de datos SQL ya que el programa los hace por ti, tambien al
 tenerlo en este programa podras dejar atras ese cuadernito en donde anotas toda la
 informacion y dejaras atras el riesgo de que dicho cuadernito se te moje o se te pierda,
 incluso la mayor ventaja de mi programa es que es seguro ya que te pide contraseña para
 poder acceder a las funciones del mismo y asi dejas atras el riesgo de que puedan alterar o
 eliminar informacion valiosa.
